# Apollo API V2

## Installing the Database

1. Install MongoDB from [here](https://www.mongodb.com/download-center?jmp=nav#community)
2. Create directory /data/db

## Running the Server

**To install**

`npm install`

**To run**

`npm start`

## Error Codes

| Code | Meaning              |
| ---- | -------------------- |
| 000  | Success              |
| 001  | Action Failed        |
| 002  | Not Found            |
| 003  | Missing/Invalid Data |
| 004  | Action Unavailable   |
